Alpine Linux 3.10.0 Released
===========================

We are pleased to announce the release of Alpine Linux 3.10.0, the first in
the v3.10 stable series.

New features and noteworthy new packages
----------------------------------------

* Support for Pine64LTS

Significant updates
-------------------
* GCC 8.3.0
* Busybox 1.30
* musl libc 1.1.22
* LLVM 8.0.0
* Go 1.12.6
* Python 3.7
* Perl 5.28
* Rust 1.34.2
* Zabbix 4.2.3
* Nextcloud 16.0.1
* Git 2.22.0
* OpenJDK 11.0.4

Credits
